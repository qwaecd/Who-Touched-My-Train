# Who Touched My Train

#### 别动我的火车，谢谢。

## 物品说明

### 列车锁 TrainLock

- 右键列车上的列车控制台即可锁定列车，当然，这辆列车必须是没有锁的，即处于```公共```权限状态。
- 当这辆车已经被上锁了，且主人是你，右键列车控制台即可在```私有```和```仅使用```之间切换权限。

|  权限等级  | 对应效果                   |
|:------:|------------------------|
|   私有   | 仅车主人或经过验证的玩家可以驾驶和使用时刻表 |
|  仅使用   | 所有人可驾驶和使用时刻表           |
|   公共   | 所有人可驾驶和使用时刻表，且可以上锁     |

### 列车钥匙 TrainKey
- 金钥匙：右键列车控制台即可解锁列车，这辆列车必须是被锁的，且玩家必须是列车的主人。
- 铁钥匙：右键该列车可以获得列车的控制权，没有时间限制，直至列车被方块化。如果是车主，则将该钥匙刻印，允许你交给其他玩家来获取这辆车的驾驶权。

## 注意事项
- 前置 mod 为 create 0.5.1.j，不兼容 6.0 及以上版本。
